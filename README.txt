Cálculo para una matriz cuadrada. Utiliza la teoría de determinantes, cofactores y adjunta, dependiendo de la implementación.

"MatrizInversa.java": clase principal que contiene los métodos para calcular la matriz inversa.
"main.java": para leer una matriz, llamar al cálculo de la inversa y mostrar o escribir el resultado.

Método de cofactores
1. Se calcula el determinante.
2. Se calcula la matriz de cofactores.
3. Se obtiene la matriz adjunta.
4. Se multiplica la matriz adjunta.
La matriz inversa de una matriz \(A\) es otra matriz, tal que al multiplicarlas se obtiene la matriz inversa.

Esto solo es posible si la matriz cumple dos condiciones:
1. Es cuadrada (mismo número de filas y columnas).  
2. Su determinante es diferente de cero.

Si el determinante vale 0 no tiene inversa.
